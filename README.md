# Production Data Analysis Dashboard (Excel)

## Project Overview
This project focuses on analyzing production data using Microsoft Excel. An interactive dashboard was created to visualize production performance across regions, managers, and product types.

## Dataset Columns
The dataset includes the following columns:
Product ID  
Date  
Region  
Manager  
Product Type  
Units Produced  
Total Cost  
Gender  
True Age  
Age Group  
Production Cost per Unit  

## Tools Used
Microsoft Excel  
Pivot Tables  
Data Visualization Charts  

## Key Features
- Data analysis using Pivot Tables
- Created and formatted multiple charts for data visualization
- Developed an interactive Excel dashboard

## Visualizations Used
- Column Chart
- Bar Chart
- Line Chart
- Area Chart
- Pie Chart

## Insights from Dashboard
- Comparison of production units across regions
- Analysis of production costs and product types
- Identification of trends in production over time
